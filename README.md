Library Management System in C# usign Windows Forms. It is for both users and admins/librarians.
Main Features:

For Students
1. User Sign Up and Login
2. Issue a Book
3. Return a Book
4. Generate Fine, maintain in Student's Profile and Send notification. 

For Admin/Librarian
1. Add a Book
2. Edit a Book
3. Delete a Book

Individual codes are also written in different cs files and collectively everything is implemented in the windowsformapplication4 folder.
